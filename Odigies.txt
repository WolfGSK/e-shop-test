To test περιλαμβάνει την υλοποίηση της μακέτας που περιλαμβάνεται στην εικόνα,
μέσα στον κώδικα που υπάρχει στο sample.html.
Συγκεκριμένα, καλείστε να κάνετε την δική σας responsive υλοποίηση μέσα στο sample, 
με html, css με <style> και javascript με <script>, inline στην γραμμή 1000, 
χρησιμοποιώντας και τα γραφικά που παρέχονται στο test.
Επισης, το html των περιοχών 3 και 4 (Α Γυμνασίου και Student home), πρέπει να γίνεται load
με Ajax, μετά από physical action του χρήστη, scroll, cursor move, touch.
Τέλος, σε κάθε κλήση θα πρέπει να δημιουργείται ένα cookie με όνομα pageload που θα αποθηκέυει 
το τρέχων timestamp.
Η σελίδα φορτώνει Jquery, που μπορεί να γίνει χρήση του ή με χρήση plain Javascript.



Χαρακτηριστικά σελίδας sample

Το responsiveness στηρίζεται στο πλάτος του viewport, αποκλειστικά, και όχι στα headers της κλήσης, 
άρα θα πρέπει να συνεχιστεί αυτή η λογική.
Η σελίδα λειτουργεί μέχρι 500 pixel σε mobile version και πάνω από 500 pixel σε desktop version, 
πάντα αναφερόμενοι στο viewport της, και σε pixel ratio 1.0. 
To πλάτος του content πρέπει να προσαρμόζεται καλύπτωντας όλο τον main container, 
τουλαχιστον από 300px μέχρι τα 1600px με pixel ratio 1.0, 
όπως είναι βασισμένη η μακέτα. Μετά τα 1600px, μπορεί να υπάρξει αναλογικό padding, 
ώστε τα elements του container που θα χτίσετε, να μην μεγαλώσουν υπερβολικά και να είναι λογικά σε εμφάνιση.

Για την επιτυχή εμφάνιση των μικρότερων αναλύσεων (mobile), μπορείτε να προσαρμόσετε το μέγεθος και των αριθμό των boxes
σε κάθε σειρά ή ακόμα και να υλοποιηθεί slider για τις περιοχές 1,3,4, καθώς είναι κυριότερο το
σωστό responsiveness, αντί για την strict αποτύπωση της μακέτας.
(Η μακέτα δεν είναι σωστή σχεδιαστικά αλλά δημιουργήθηκε μόνο για το test)



Hints

- Το μέγεθος των elements δεν πρέπει να είναι δυσανάλογο με τον υπόλοιπο σκελετό της σελίδας.
- Το responsiveness πρέπει να λειτουργεί σωστά, ώστε να μην σπάει το πλάτος του εκάστοτε τρέχωντος viewport, 
οπότε και να μην εμφανίζει οριζόντια scrollbar σε καμία ανάλυση.
